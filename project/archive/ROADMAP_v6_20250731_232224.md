## Technical Stack
- **Frontend:** Next.js 14 (App Router), TypeScript, Tailwind CSS
- **Backend:** Supabase (PostgreSQL, Auth, Storage, Edge Functions)
- **State:** Zustand + React Query
- **Maps:** Mapbox GL JS
- **Deployment:** Vercel + Supabase
